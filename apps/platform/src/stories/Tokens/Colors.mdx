# Color Tokens

Keyshade uses a combination of Tailwind CSS colors and custom colors to create a unique color palette.
Our team decided to use the OKLCH color space for its perceptual uniformity and accessibility benefits. to read more about OKLCH, check out [this article](https://evilmartians.com/chronicles/oklch-in-css-why-quit-rgb-hsl).

## Tailwind Colors

We use Tailwind CSS base colors as our primary color palette. These colors are designed to be accessible and work well together.
